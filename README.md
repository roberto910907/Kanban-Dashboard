## Tech Stack
- Laravel v9
- Nuxt v3
- VueJS 3
- GraphQL
- TailwindCSS

## Structure
- **api** folder: Laravel 9 application serving a GraphQL endpoint
- **app** folder: Nuxt 3 application serving frontend
- **docs** folder: Nuxt Docus serving project documentation
